01 / WebGL + GLSL

PARTICLE
FIELD

The background is a live WebGL canvas rendering thousands of particles via custom GLSL vertex & fragment shaders. Mouse position is passed as a uniform — particles react to your cursor in real time.

Technologies: gl_PointSize · uniform mat4 · float noise · gl_FragCoord

// GLSL Vertex Shader
void main() {
  vec3 p = position;
  float t = uTime * 0.5;
  p.x += sin(p.y*2.+t)*15.;
  p.y += cos(p.x*2.+t)*10.;
  
  // Mouse repulsion
  vec2 d = p.xy - uMouse;
  float dist = length(d);
  if(dist < 80.) {
    p.xy += normalize(d)
          * (80.-dist) * 0.3;
  }
  gl_PointSize = 2.;
  gl_Position = projMat 
    * viewMat * vec4(p,1.);
}
